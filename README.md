# dump

Personal grab-bag repo. Layout by maturity:

- **`done/`** — projects that work and are mostly finished.
- **`playground/`** — experiments, toys, WIP, one-offs.
- **`scripts/`** — install/setup scripts for my machine (`install_*.bash`).
- **`lcc/`** — university coursework (Licenciatura en Cs. de la Computación).

## done/
- `facebook_scrapper/` — scrape engagement data from Facebook post URLs.
- `wallpaper_manager/` — set image/video wallpapers via C + ffmpeg/cairo.
- `corne_keyboard/` — QMK firmware build + config for my Corne keyboard.
- `typerl/` — terminal typing trainer, Perl + Curses, configurable.

## playground/
- `coding_problems/` — LeetCode, Advent of Code, Perl Weekly Challenge, misc.
- `fish_tank/` — terminal fish-tank animation.
- `save3rdGen/` — reader for 3rd-gen Pokémon `.sav` files.
- `thimbleweed_park_bin_reader/` — decode a binary blob from Thimbleweed Park.
- `hello_world_bug/` — a hello-world that reproduces a bug.
- `scripts/` — assorted one-off scripts (battery, wallpaper cycling, letterboxd...).
