# corne_keyboard

QMK firmware for my Corne (crkbd) v2 keyboard.

- `corne_firmware.sh` — installs QMK and builds/flashes the firmware.
- `crkbd_rev1_v2_simple.json` — layout config.
- `crkbd_rev1_v2_simple.hex` — compiled firmware, flash with `qmk --flash <hex>`.

Build notes and guide links are in the header of `corne_firmware.sh`.
